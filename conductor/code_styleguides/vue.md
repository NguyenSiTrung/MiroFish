# Vue.js Style Guide (Frontend)

## Conventions (inferred from codebase)

- Vue 3 with `<script setup>` Composition API (SFC pattern)
- Vite as build tool with `@vitejs/plugin-vue`
- ES module syntax (`import`/`export`)
- Vue Router for client-side routing (`src/router/`)
- State management via `src/store/`
- API layer abstracted in `src/api/`
- Component organization in `src/components/`, page-level views in `src/views/`
- D3.js for data visualization components
- Axios for HTTP requests
- Global CSS reset in `App.vue`, component-scoped styles preferred
- Font stack: JetBrains Mono, Space Grotesk, Noto Sans SC
- Black and white color scheme as base theme
